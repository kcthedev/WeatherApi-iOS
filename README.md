# Weather API

[![Swift6_3_2](https://img.shields.io/badge/Swift-6.3.2-ff69b4)]() [![SwiftUI](https://img.shields.io/badge/SwiftUI-lightgray)](https://developer.apple.com/swiftui/) [![MVVM](https://img.shields.io/badge/MVVM-red)]() [![Navigation](https://img.shields.io/badge/Navigation-2.8-green)]() [![URLSession](https://img.shields.io/badge/URL_Session-blue)]() [![Serialization](https://img.shields.io/badge/Serialization-purple)]()

Simple demo project that gets weather data for a given city from an external API and displays it asynchronously in the Swift UI using async functions. Supports a home screen where the user can see weather data for a list of cities and a details screenm, which provides additional weather infofor a particular city, as well as navigation between them. Maintains clean architecture and unit tests for critical business logic.

**Features**
----------

- Retrieve data from remote source ✔️
- Display data in Swift UI ✔️
- Details screen for most in-depth info on particular city ✔️
- Navigate between screens with SwiftUI Navigation ✔️
- Live updates with async functions ✔️
- Clean Architecture ✔️
- Supports both Portrait and Landscape orientation ✔️
